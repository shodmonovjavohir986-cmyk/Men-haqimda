# 🤖 JSH Portfolio Bot

JSH Portfolio Bot — bu mening portfolio va xizmatlarimni taqdim etuvchi Telegram bot.

## ✨ Imkoniyatlari

- 👨‍💻 Men haqimda ma'lumot
- 📂 Portfolio loyihalarini ko'rish
- 🛠 Ko'nikmalar va texnologiyalar
- 📞 Bog'lanish ma'lumotlari
- 🔗 Ijtimoiy tarmoqlar havolalari

## 🚀 Texnologiyalar

- Python
- Aiogram
- Telegram Bot API

🌐 **Frontend:**
• HTML5
• CSS3
• JavaScript    

## 📦 O'rnatish

1. Loyihani yuklab oling:
```bash
git clone <repository_link>
```

2. Kerakli kutubxonalarni o'rnating:
```bash
pip install -r requirements.txt
```

3.`main.py` fayliga bot tokenini kiriting.

4. Botni ishga tushiring:
```bash
python main.py
```

## 📌 Maqsad

Ushbu bot orqali foydalanuvchilar mening portfolio loyihalarim, xizmatlarim va aloqa ma'lumotlarim bilan tez va qulay tanishishlari mumkin.
